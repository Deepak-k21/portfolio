## ✨ Features

- 🎭 **Interactive 3D Character** — Custom animated character built with Three.js & React Three Fiber
- 🌊 **Smooth Scroll Animations** — Powered by GSAP ScrollSmoother & ScrollTrigger
- 🖼️ **Horizontal Project Gallery** — Scroll-triggered horizontal carousel showcasing deployed client projects
- ⚡ **Physics-Based Tech Stack** — Interactive bouncing tech spheres using Rapier physics engine
- 🎯 **Custom Cursor** — Dynamic cursor that responds to interactive elements
- 📱 **Fully Responsive** — Optimized for all screen sizes from mobile to 4K
- 🚀 **Deployed Projects** — Every project card links to a live production website

## ⚙️ Getting Started

```bash
# Clone the repository
git clone https://github.com/Deepak-k21/portfolio.git

# Navigate to the project
cd portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be running at `http://localhost:5173/`

## 📁 Project Structure

```
portfolio/
├── public/
│   ├── images/          # Project screenshots
│   ├── models/          # 3D model files (.glb, .hdr)
│   └── draco/           # Draco decoder for compressed 3D models
├── src/
│   ├── components/      # React components
│   │   ├── Character/   # 3D character model & animations
│   │   ├── styles/      # Component-specific CSS
│   │   └── utils/       # Utility functions
│   ├── context/         # React context providers
│   ├── data/            # Static data files
│   └── assets/          # Static assets
└── index.html
```
